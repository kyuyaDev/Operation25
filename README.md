# 🚀 연산 25 (Operation 25)

[연산 25 - 게임하기](https://operation25.netlify.app/)

**연산 25 (Operation 25)**는 25라운드 동안 펼쳐지는 하드코어 수학 연산 게임입니다. 단순한 계산을 넘어, 빠르고 정확한 두뇌 회전을 요구하며 전 세계 플레이어들과 기록을 경쟁할 수 있습니다.

---

## ✨ 주요 기능 (Key Features)

- **🔥 25라운드 챌린지**: 덧셈, 뺄셈, 곱셈, 나눗셈이 혼합된 25개의 문제를 중단 없이 풀어야 합니다.
- **💀 하드코어 모드**: 힌트나 패스 기능이 없습니다. 오답 시 문제를 맞출 때까지 다음 라운드로 넘어갈 수 없습니다.
- **🎨 3종 테마 지원**: 
  - **Space Night**: 깊은 우주의 밤을 연상시키는 다크 모드.
  - **Classic Chalk**: 정겨운 학교 칠판 느낌의 테마.
  - **Cyber Grid**: 미래지향적인 사이버펑크 스타일.
- **🎉 역동적인 연출**: 정답 시 화려한 폭죽(Confetti) 효과와 오답 시 강렬한 화면 흔들림(Intense Shake) 피드백.
- **🔊 사운드 컨트롤**: BGM과 SFX를 통합 제어할 수 있는 사운드 토글 기능.
- **🏆 실시간 랭킹**: Supabase를 연동하여 최고 기록을 실시간으로 확인하고 경쟁할 수 있습니다.

---

## 🛠 기술 스택 (Tech Stack)

- **Language**: Vanilla JavaScript (ES6+)
- **Styling**: CSS3 (Custom Variables, Keyframes, Flex/Grid)
- **Database**: Supabase (PostgreSQL)
- **Audio**: Web Audio API (Dynamic SFX & BGM Engine)
- **Effects**: Canvas Confetti API

---

## 🎮 게임 플레이 (How to Play)

1. **게임 시작**: 시작 화면에서 '게임 시작' 버튼을 누릅니다.
2. **정답 입력**: 가상 키패드나 물리 키보드 숫자를 사용하여 정답을 입력합니다.
3. **정답 확인**: '입력' 버튼 또는 'Enter' 키를 눌러 제출합니다.
4. **목표**: 최대한 빠르게 25라운드를 완료하고 자신의 이름을 랭킹에 등록하세요!
